poolRestriction 

case class poolRestriction(poolNames: List[String], argument: String, limit: Int) extends Restriction with Product with Serializable

A restriction for pool values, uses common comparators to determine if an entity meets the restriction.

All params subject to change, work in progress.

poolNames

A list of applicable pool names against which the restriction can be checked.

argument

The comparator to use i.e. ">", "==","!=","<=", etc...

limit

The value against which to compare the current pool value.

Version

0.1
